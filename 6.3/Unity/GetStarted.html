开始集成

开始集成

须知

  • iOS平台： Unity Plugin支持iOS8 支持Unity 4和5.4.1或更高版本 Vungle SDK需要依赖WebKit.framework。如果您要支持iOS7,请务必设置该依赖为'Optional'。
  • Android平台： Unity Plugin依赖Java 1.7 支持Unity 4和5.3.2或更高版本
  • Windows平台： 支持Unity 4和5.3.2或更高版本 Windows 10 UWP 支持 Unity 5.3.2+
  • Unity 2018 目前还不支持

集成Plugin

下载Unity Plugin，解压缩并双击VunglePlugin.unitypackage，点击Import Unity Package进行导入。

初始化Vungle Plugin

在请求广告之前，请先初始化SDK。一般来说需要在App启动时立刻初始化。 初始化成功之前，load广告无效。 示例：

public class VungleScript : MonoBehaviour {

    void Start(){
        string appID = "";
        string iosAppID = "ios_app_id";
        string androidAppID = "android_app_id";
        string windowsAppID = "windows_app_id";
        #if UNITY_IPHONE
            appID = iosAppID;
        #elif UNITY_ANDROID
            appID = androidAppID;
        #elif UNITY_WSA_10_0 || UNITY_WINRT_8_1 || UNITY_METRO
            appID = windowsAppID;
        #endif
        Vungle.init(appID);
    }

    void OnApplicationPause(bool pauseStatus)
    {
        if (pauseStatus)
        {
            Vungle.onPause();
        }
        else
        {
            Vungle.onResume();
        }
    }
}

GDPR

《通用数据保护条例》(GDPR) 将于2018年 5 月 25 日在欧盟生效。为了遵守 GDPR，Vungle提供了如下方法。 The consentMessageVersion indicates publisher controlled consent policy version to allow publisher getting consent again when GDPR policy changes by pooling users by the message version.

// Usage example of GDPR API
// Vungle.Consent.Accepted 允许使用用户私有信息投放广告
// Vungle.Consent.Denied 不允许使用用户私有信息投放广告
Vungle.updateConsentStatus(Vungle.Consent.Accepted);
Vungle.updateConsentStatus(Vungle.Consent.Denied);

// To find out what the user's current consent status is:
Vungle.Consent currentStatus = Vungle.getConsentStatus();
