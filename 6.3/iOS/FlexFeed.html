Flex Feed

Flex Feed

Flex Feed又名原生广告，用户可传递view来自定义广告播放窗口大小。

请求广告

SDK发送广告请求，在服务器给填充后开始缓存视频广告素材。Placement Reference id传送门​

NSError* error;
VungleSDK* sdk = [VungleSDK sharedSDK];
[sdk loadPlacementWithID:"YOUR_FLEX_FEED_PLACEMENT_ID" error:&error];

播放广告

VungleSDK* sdk = [VungleSDK sharedSDK];
// 静音播放
sdk.muted = true;
if([sdk isAdCachedForPlacementID:@"YOUR_RWARD_PLACEMENT_ID"]){
    NSError* error;
    [sdk addAdViewToView:_mView withOptions:nil placementID:flexFeed error:&error];
    if (error) {
        NSLog(@"Error encountered playing ad: %@", error);
    }
}

关闭广告

由于用户可以在不关闭广告的情况下，停止观看广告（例如：划出广告播放区域）。

VungleSDK* sdk = [VungleSDK sharedSDK];
[sdk finishedDisplayingAd];

Flex Feed广告提供了可关闭广告接口。但由于同一时间只能播放一个Flex Feed类型广告，因此关闭时并未指定Placement Id。
