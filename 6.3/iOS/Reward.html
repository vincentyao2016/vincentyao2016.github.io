Reward奖励广告

Reward 奖励广告

Vungle的奖励广告是全屏视频广告。在播放完毕广告后，会自动跳转到落地页。在落地页用户可点击跳转到商店下载广告应用或游戏。

请求广告

SDK发送广告请求，在服务器给填充后开始缓存视频广告素材。Placement Reference id传送门​

NSError* error;
VungleSDK* sdk = [VungleSDK sharedSDK];
[sdk loadPlacementWithID:"YOUR_RWARD_PLACEMENT_ID" error:&error];

播放广告

VungleSDK* sdk = [VungleSDK sharedSDK];
// 静音播放
sdk.muted = true;
if([sdk isAdCachedForPlacementID:@"YOUR_RWARD_PLACEMENT_ID"]){
    NSError* error;
    [sdk playAd:self options:nil placementID:@"YOUR_RWARD_PLACEMENT_ID" error:&error];
    if (error) {
        NSLog(@"Error encountered playing ad: %@", error);
    }
}
