Interstitial 插屏

Interstitial 插屏

Vungle插屏广告指的可跳过的视频广告。其广告素材与奖励广告是相同的。插屏与奖励广告取决于传入参数placement reference id在dashboard上的类型。

请求广告

SDK发送广告请求，在服务器给填充后开始缓存视频广告素材。Placement Reference id传送门。

// Load Ad 
Vungle.loadAd("YOUR_INTERSTITIAL_PLACEMENT_REFERENCE_ID");

播放广告

if( Vungle.isAdvertAvailable("YOUR_INTERSTITIAL_PLACEMENT_REFERENCE_ID")){
   // 静音播放
   Vungle.setSoundEnabled(false);  
   Vungle.playAd("YOUR_INTERSTITIAL_PLACEMENT_REFERENCE_ID");
}
