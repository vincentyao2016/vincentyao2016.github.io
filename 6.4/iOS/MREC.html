MREC

MREC广告

SDK 6.4.8及以上才支持MREC广告。MREC采用移动广告行业的标准样式，长宽为300dp × 250dp。和FlexFeed相同，可以将该广告在任意位置展示。

请求广告

SDK发送广告请求，在服务器给填充后开始缓存视频广告素材。Placement Reference id传送门。

NSError* error;
VungleSDK* sdk = [VungleSDK sharedSDK];
[sdk loadPlacementWithID:"YOUR_MREC_PLACEMENT_ID" error:&error];

播放广告

private ConstraintLayout flexfeed_container;
private VungleNativeAd vungleNativeAd;
private View nativeAdView;

...
NSError *error;
NSDictionary *options = @{};
VungleSDK* sdk = [VungleSDK sharedSDK];
If (![sdk addAdViewToView:_mrecViewArea withOptions:options placementID:"YOUR_MREC_PLACEMENT_ID" error:&error]) {
    if (error) 
    {
        NSLog(@"Error encountered while playing an ad: %@", error);
    }
}

关闭广告

//Ad is no longer on screen
VungleSDK* sdk = [VungleSDK sharedSDK];
[sdk finishedDisplayingAd];

注意

如果在Dashboard设置了auto cache则无需手动调用load方法来加载广告。
SDK会在初始化成功后，自动加载第一条广告。
SDK会在播放时，自动加载下一条。
自动缓存成功后，会在初始化中有回调onAutoCacheAdAvailable。
