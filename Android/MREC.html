MREC

MREC广告

SDK 6.4.8及以上才支持MREC广告。

请求广告

SDK发送广告请求，在服务器给填充后开始缓存视频广告素材。Placement Reference id传送门。

if (Vungle.isInitialized()) {
    Vungle.loadAd("YOUR_MREC_PLACEMENT_ID"", new LoadAdCallback() {
        @Override
        public void onAdLoad(String placementReferenceId) { }
​
        @Override
        public void onError(String placementReferenceId, Throwable throwable) {
            // Load ad error occurred - throwable.getLocalizedMessage() contains error message
        }
    });
}

播放广告

private ConstraintLayout flexfeed_container;
private VungleNativeAd vungleNativeAd;
private View nativeAdView;

...

AdConfig.setAdSize(AdConfig.AdSize.VUNGLE_MREC);
flexfeed_container = findViewById(R.id.mlayout);
vungleNativeAd =Vungle.getNativeAd("MREC_Placement_reference_id", new PlayAdCallback() {
    @Override
    public void onAdStart(String placementReferenceId) { }

    @Override
    public void onAdEnd(String placementReferenceId, boolean completed, boolean isCTAClicked) {
        // 关闭广告
        flexfeed_container.removeView(nativeAdView);
        vungleNativeAd = null;
    }

    @Override
    public void onError(String placementReferenceId, Throwable throwable) {
        // Play ad error occurred - throwable.getLocalizedMessage() contains error message
    }
});
nativeAdView = vungleNativeAd.renderNativeView();
flexfeed_container.addView(nativeAdView);

暂停播放

vungleNativeAd.setAdVisibility(false);

恢复播放

vungleNativeAd.setAdVisibility(true);
