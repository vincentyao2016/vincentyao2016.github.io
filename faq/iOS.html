iOS Q&A

上架 Missing Purpose String in Info.plist File

苹果商店要求上架应用在plist中添加使用NSLocationWhenInUseUsageDescription的描述。可以添加描述如下：ad reporting, metrics, and ad personalization。Vungle6.4版本及之后版本将移除使用该功能的部分代码，以解决该问题。

回调- (void)vungleAdPlayabilityUpdate:(BOOL)isAdPlayable placementID:(nullable NSString )placementID error:(nullable NSError )error 中error会在什么时候产生，可能都error都有哪些？

@"There are no playable ads available at this time." 调用播放时回调

sdk loadPlacementWithID:"YOUR_INTERSTITIAL_PLACEMENT_ID" error:&error]方法中的error都有哪些，什么情况下触发呢？

[NSString stringWithFormat:@"VungleSDK didn't find a matching placementID for %@", placementID]; [NSString stringWithFormat:@"VungleSDK placement with the ID %@ is sleeping", placementID]; 在placement id不匹配和被无填充被暂时停止服务时触发。该过程不会触发load回调中的error。
