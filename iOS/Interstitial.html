Interstitial 插屏

Interstitial 插屏

Vungle插屏广告指的可跳过的视频广告。其广告素材与奖励广告是相同的。插屏与奖励广告取决于传入参数placement reference id在dashboard上的类型。 插屏广告也可以设置为不可跳过。此设置在dashboard上设置。

请求广告

SDK发送广告请求，在服务器给填充后开始缓存视频广告素材。Placement Reference id传送门​

NSError* error;
VungleSDK* sdk = [VungleSDK sharedSDK];
[sdk loadPlacementWithID:"YOUR_INTERSTITIAL_PLACEMENT_ID" error:&error];

播放广告

VungleSDK* sdk = [VungleSDK sharedSDK];
// 静音播放
sdk.muted = true;
if([sdk isAdCachedForPlacementID:@"YOUR_INTERSTITIAL_PLACEMENT_ID"]){
    NSError* error;
    [sdk playAd:self options:nil placementID:@"YOUR_INTERSTITIAL_PLACEMENT_ID" error:&error];
    if (error) {
        NSLog(@"Error encountered playing ad: %@", error);
    }
}
